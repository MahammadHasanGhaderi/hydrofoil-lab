# NACA Airfoil Generator

This project is a Python implementation of the NACA 4-digit airfoil equations.  
It generates the geometry of airfoils such as NACA 0012, 4412, etc., and visualizes the upper surface, lower surface, and camber line.

## Features
- Generate NACA 4-digit airfoils
- Compute camber line and thickness distribution
- Plot airfoil geometry
- Object-oriented design for future expansion (CL/CD, flow analysis)

## Future Work
